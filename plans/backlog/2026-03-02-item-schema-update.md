# Item Schema Update

Update the core item schema to support richer metadata and relationships:

- **Title**: Short name/headline for the item
- **Description**: Longer details and notes
- **Url[]**: Multiple URLs associated with the item
- **Due date**: When the item needs to be completed
- **Priority**: Importance level (high/medium/low)
- **List[]**: Support for items belonging to multiple lists/tags

This expanded schema enables the Siri, share sheet, and URL features while making the app more flexible for different use cases.
