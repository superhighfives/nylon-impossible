# Siri Integration

Add Siri support to Nylon so users can create tasks using voice commands. Examples include "Hey Siri, tell Nylon to remind me to get dog food" or "Hey Siri, add 'feed cat' to Nylon". This would make it much faster to capture tasks when hands are busy or when an idea strikes without having to open the app.

This is a natural extension since Nylon is a task management app where quick capture is essential. The iOS app already exists and has access to SiriKit for intents.
